# ADR-0006: 確定反撃の双方向化と全キャラOoSデータ保持

Status: Accepted (2026-07-03)

## Context
リサーチ（docs/04）で先行事例 outofshield.com が確認された。同サイトは「相手技→確反」に加え、逆引き「自分のどの技が相手のガードに安全か(safe on shield)」を提供しており、攻め側の技選択にも同じデータが使える。ultimateframedata.com はキャラ別OoS発生一覧を公開しており、全キャラ分のOoSデータ取込が可能。

## Decision
- `oos_options` をZSS専用から**全キャラ分**に拡張。UFDのOoSデータから自動取込（ZSS含む。代表値のスポットチェックのみ行い、誤りに気づいた時に修正）
- 確定反撃機能を双方向にする:
  - **守り**: 相手の技を選ぶ → ZSSの確定反撃リスト（従来設計）
  - **攻め**: ZSSの技を選ぶ → 相手キャラのOoS実効発生と比較し、ガードさせて安全/反撃確定を判定（キャラ対ページに「この技は振っていいか」ビュー）

## Consequences
- 同一データ・同一計算式で機能が2倍になる（実装増分は小さい）
- 全キャラOoSの初期取込が増えるが、UFDに一覧があるため機械的に可能
- 判定はどちらの向きでも「フレーム上」の結論であり、リーチ/間合いの注記は必須（outofshield.com自身の教訓）
