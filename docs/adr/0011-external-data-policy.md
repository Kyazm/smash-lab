# ADR-0011: スマメイト自動連携の見送りと外部サイトはリンク参照

Status: Accepted (2026-07-03)

## Context
調査（.context/research-smashmate.md, research-pheasantzelda.md）の結論:
- スマメイト: 公式APIなし。対戦履歴詳細はログイン必須の可能性が高く、先行スクレイピングツールは仕様変更で停止した前例あり
- pheasantzelda（スマアナ）: スマメイト戦績約1880万戦のマッチアップ統計・キャラランクのポータル。LICENSEファイルがなくデータ複製の法的根拠が不明瞭

## Decision
- スマメイトの自動取込は**作らない**。プレイヤー別メモにスマメイトID/URLを手動記録する運用（notes.player_name + 本文）
- pheasantzeldaの統計データは**転載しない**。キャラ対ページに同サイトのMUページへの外部リンクを自動生成して1タップで参照できるようにする（UFDのキャラページリンクも同様）
- スマメイトにAPIが提供されたら再評価

## Consequences
- 継続運用リスク（仕様変更での破損、規約・ライセンス問題）をゼロにしつつ、統計情報へは1タップで到達可能
- MU勝率などをアプリ内で横断集計することはできない（許容）
