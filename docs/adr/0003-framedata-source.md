# ADR-0003: フレームデータはultimateframedata.comから一括取込

Status: Accepted (2026-07-03)

## Context
スマブラSPはVer.13.0.1（2021年）でバランス調整終了。フレームデータは静的で、一度取り込めば更新不要。ultimateframedata.comは全キャラ網羅で発生/全体F/ガード硬直差/ヒットボックス画像が揃う事実上の標準。日本語ソースは構造が不揃いで、硬直差の掲載もまちまち。

## Decision
ultimateframedata.comをスクレイプして一括取込。日本語技名は検証窓スプレッドシート（ユーザー判断で信頼ソースに確定）から取得し、不足分のみAI生成→人間確認。ヒットボックス画像は自前ストレージへミラー。対象はスマブラSPのみ。

## Consequences
- 全キャラの硬直差データが確定反撃判定(F3)にそのまま使える
- 個人利用のスクレイプだが、画像ミラーで先方への継続負荷を避ける
- 代替案: GitHubに検証済みJSONがあれば取込元を差し替え可（実装計画のリスク欄参照）
