# ADR-0009: 統一キャラページと /me の廃止

Status: Accepted (2026-07-03)

## Context
自キャラ専用ページ `/me` はフレーム表・確定反撃・キャラ対メモ（ミラー対策）を持たず、他キャラページと機能が分裂していた。ミラーマッチの対策メモは既に `matchup/zero_suit_samus` として存在するが表示先がなかった。また `/me` の立ち回りメモは18件が縦一列で長すぎた（ユーザー指摘）。

## Decision
- 全キャラを `/c/:slug` の統一ページにし、タブ（URLパラメータ `?tab=`）で切替: `frames | punish | notes`（共通）+ `own | moves`（is_mainのみ）
- `/me` は自キャラページへリダイレクト（slugはハードコードせず `characters.is_main` から動的解決。使用キャラ変更に耐える）
- 立ち回りタブはDiscord元カテゴリから復元したタグ（ゼロサム/基本）でフィルタし、メモはデフォルト折りたたみ
- タブ・フィルタ状態はURLに持つ（共有・ブラウザバック対応）

## Consequences
- ミラー対策・自キャラのフレーム/確反参照が可能になる。ページ実装が1系統に統一され保守が楽
- ZSSページはタブ5個になるが、横スクロール可能なセグメントコントロールで吸収
